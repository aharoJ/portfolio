# How To Run 

## Frontend 
1. Clone 
```sh
git clone https://github.com/aharoJ/portfolio.git
```

2. CD into path

../../portfolio

3. clean and install dependencies

```sh
npm install --force
```

4. Run in your local enviroment

```sh
 npm run dev
```
